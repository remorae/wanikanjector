Wanikanjector Options

Public API Key

Your API key can be obtained from your WaniKani account page. This key is required to retrieve your vocabulary list.

Vocabulary Cache

Your WaniKani vocabulary is stored and updated hourly (if needed) for performance reasons.

Run On Load

Check to replace text when a page loads. You can toggle text for a page at anytime via the popup menu.

SRS Filters

Check levels to include their kanji.

Custom Vocabulary

Enter words here to override WaniKani vocabulary.
Example: dog,terrier;犬;いぬ
One entry per line.

Note the commas between English synonyms and semicolons before the vocab/reading.
Black List

Prevent Wanikanjector from running automatically on any matching sites. Sites will be blacklisted if any pattern matches the URL.

You may use regular expressions (regex). For more help, see here.

These are the current browser permissions required by Wanikanjector.
