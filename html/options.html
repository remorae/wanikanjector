Wanikanjector Options

Public API Key

Your API key can be obtained from your WaniKani account page. This key is required to retrieve your vocabulary list.

Vocabulary Cache

Your WaniKani vocabulary is stored locally for performance reasons (and to play nice with WaniKani's servers). Click this button to clear the cache.

Run On Load

You can have text replacement occur when the page loads. If disabled, you can run when the button is clicked via the popup menu.

SRS Filters

Only words from the checked groups will be substituted with kanji.

Custom Vocabulary

Enter words here to override WaniKani vocabulary. Format: english1,english2,...,vocab;reading; Example: cat,feline;猫;ねこ One entry per line. Note the commas between English synonyms as well as the semicolons before the kanji/reading.

Black List

Enter sites below to prevent Wanikanjector from running on them automatically regardless of the above setting. Sites can be entered using regular expressions (regex). A helpful site for creating regex can be found here. Sites will be blacklisted if any pattern matches the URL. Make sure to click save after making any changes to the list!
