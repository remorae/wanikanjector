Wanikanjector Options

WaniKani API Token

Your API token can be obtained from your WaniKani API tokens page. This is required to retrieve WaniKani vocabulary accessible to you based on your subscription, as well as what items you've learned. A read-only token is sufficient.

Vocabulary Cache

Your WaniKani vocabulary is stored and updated hourly (if needed) for performance reasons.

Run On Load

Check to replace text when a page loads. You can toggle text for a page anytime via the popup menu.

Replaced Vocabulary

Uncheck to replace text with all the vocab in Wanikani, not just vocab that you have learned or is available at your level. You will need to clear the cache after changing this setting.

Uncheck to replace all meanings for a particular word rather than only the primary meaning.

SRS Filters

Check levels to include their kanji.

Custom Vocabulary

Enter words here to override WaniKani vocabulary.
Example: dog,terrier;犬;いぬ
One entry per line.

Note the commas between meanings and semicolons before the vocab/reading.

Blacklist

Prevent Wanikanjector from running automatically on any matching sites. Sites will be blacklisted if any pattern matches the URL.

You may use regular expressions (regex). For more help, see here.

Permissions

These are the current browser permissions required by Wanikanjector.
