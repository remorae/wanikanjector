Wanikanjector Options

Public API Key

Your API key can be obtained from your WaniKani account page. This key is required to retrieve your vocabulary list.

Vocabulary Cache

Your WaniKani vocabulary is stored locally for performance reasons (and to play nice with WaniKani's servers). Click this button to clear the cache.

Run On Load

You can have text replacement occur when the page loads. If disabled, you can replace text by clicking the button in the popup menu.

SRS Filters

Only words from the checked groups will be substituted with kanji.

Custom Vocabulary

Enter words here to override WaniKani vocabulary.
Example: dog,terrier;犬;いぬ
One entry per line.

Note the commas between English synonyms as well as the semicolons before the vocab/reading.

Black List

Prevent Wanikanjector from running automatically on any matching sites. Sites will be blacklisted if any pattern matches the URL.

You may use regular expressions (regex). For more help, see here.
